# EMPS6 Developer Bootcamp

Here's a collection of markdown documents for new developer onboarding.
Although EMPS6 prjects use quite commonly used tools and frameworks, such as
Smarty, Vue, PHP/MySQL, Composer, ffmpeg, etc., sometimes the way it uses
those tools may seem unconventional to outsiders.

The blocks below are links to other .md documents on various topics,
feel free to add your insights to this documentation if you feel that it's
missing something important.

> **For AI assistants (Claude etc.):** Before starting work on an EMPS project,
> also read the files in the `../Claude/` folder — they contain concise
> reference material on routing, patterns, and common gotchas specifically
> compiled for efficient AI-assisted development.
